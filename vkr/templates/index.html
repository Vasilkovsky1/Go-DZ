Mini DEV.to — это учебный аналог платформы DEV.to. Здесь пользователи могут регистрироваться, авторизоваться, публиковать статьи и читать материалы других авторов.

Возможности сайта

  • Регистрация пользователей
  • Авторизация
  • Создание и публикация статей
  • Просмотр статей других пользователей
  • Форма обратной связи

Backend на Go

Сервер обрабатывает маршруты, формы, авторизацию и работу с базой данных.

SQLite

В базе данных хранятся пользователи, статьи и сообщения из формы контактов.

Frontend

Интерфейс реализован с помощью HTML, CSS и JavaScript.
